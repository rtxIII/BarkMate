// curl template — copy & ship
curl -X POST "https://api.day.app/<key>" \
  -d group=backend-refactor \
  -d task_id=auth-migration-0420 \
  -d agent_status=running \
  -d progress=3/8

task_id→ 聚合键 · 决定是否原地更新
agent_statusrunning · waiting_input · blocked · done · failed
