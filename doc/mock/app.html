— One-line install —

One script.
Every agent wired.

跑一遍这段脚本，它会用你的 device key 自动配好 Claude Code / Codex / OpenCode 的 hook，自建 agent 留一个通用 bark-push 命令兼容。

// detects ~/.claude, ~/.codex, ~/.opencode and installs hooks
curl -fsSL "https://barkagent.we2.xyz/install.sh" \
  | BARK_KEY=KXHpCVR-7bygYOAHhXNsIg sh

// or for one specific agent
barkmate install --agent=claude --key=$BARK_KEY
// supported: claude · codex · opencode · custom
claude→ ~/.claude/settings.json · SessionStart + Stop + Notification
codex→ ~/.codex/hooks.toml · on_start + on_complete + on_block
opencode→ ~/.opencode/agents/*.yaml · event:* webhook
custom→ /usr/local/bin/bark-push · POST agent_status/task_id/progress
